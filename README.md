# 🎮 Juego de Palabras - Proyecto ADA

Este repositorio contiene un juego de palabras en Python desarrollado como trabajo práctico para el curso de programación en ADA - Mujeres en Tecnología.

## 🎥 Video demostrativo

[![Ver en YouTube](https://img.youtube.com/vi/nwrOw3aJN3A/0.jpg)](https://youtu.be/nwrOw3aJN3A)




## 🧩 Descripción

El juego consiste en adivinar una palabra secreta, letra por letra o arriesgando la palabra completa. El jugador tiene un número limitado de vidas, y el juego incluye mensajes interactivos y validaciones de entrada.
